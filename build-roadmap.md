# Build Roadmap

## Phase 1 - Sellable Manual MVP

Goal:

Create a reliable customer-specific setup using GREEN API, Make, Google Calendar, and Google Sheets/Airtable.

Deliverables:

- Demo WhatsApp flow
- Onboarding form
- Make scenario template
- Customer configuration sheet
- Sales page or PDF offer
- 3 pilot customers

Success metric:

- Each pilot gets at least one measurable result: saved lead, confirmed appointment, prevented no-show, or review request sent.

## Phase 2 - Repeatable Implementation Kit

Goal:

Reduce setup time per customer.

Deliverables:

- Standard payload library
- Reusable message templates
- Industry presets
- QA checklist
- Scenario duplication instructions
- Pricing tiers

Success metric:

- New customer can be configured in under 2 hours after account access.

## Phase 3 - SaaS Dashboard

Goal:

Move from Make-heavy implementation to a central dashboard.

Deliverables:

- Customer admin dashboard
- Service editor
- Message editor
- Appointment and lead dashboard
- Webhook backend
- GREEN API instance manager
- Reports

Success metric:

- Non-technical operator can onboard a customer without editing Make scenarios.

## Phase 4 - Scale

Goal:

Turn it into a productized service or SaaS.

Deliverables:

- Landing page
- Demo booking funnel
- Partner/referral program
- White-label installer guide
- Support process
- Billing automation

Success metric:

- 20 paying customers with repeatable onboarding and support load under control.
